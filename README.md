sl-mac
======

Port Masashi Toyoda's SL (Steam Locomotive) to Mac OS X


##Install Steps

Clone

```bash
$ git clone https://github.com/Fykec/sl-mac.git
```

Install

```bash
$ cd sl-mac
$ make
$ sudo cp sl /usr/local/bin && chmod a+x /usr/local/bin/sl #need input your password
```

##Usage

```base
$ sl
$ sl-l
$ sl-h
```
